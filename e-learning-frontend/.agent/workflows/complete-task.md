# Hoàn thành task

> Dùng workflow này sau khi xong một task để cập nhật tiến độ.

---

## Bước 1 — Xác nhận task đã xong

Kiểm tra checklist trong workflow tương ứng đã tick hết.

---

## Bước 2 — Cập nhật QLCV/QLCV-FE.md

Thay đổi trạng thái task từ `⬜ Todo` sang `✅ Done` và ghi ngày hoàn thành.

---

## Bước 3 — Cập nhật tổng tiến độ

Cập nhật bảng tổng tiến độ ở cuối file `QLCV/QLCV-FE.md`:
- Số task hoàn thành
- Phần trăm %

---

## Bước 4 — Gợi ý task tiếp

Dựa vào lộ trình sprint và dependencies BE/FE, gợi ý task tiếp theo.
